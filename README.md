
# 🌿 Debloat Daily

A free offline PWA habit tracker to help reduce face and stomach bloating through simple daily routines.

## Live App
👉 https://your-site.netlify.app

## Features
- ✅ Daily checklist with morning to night routine
- 📅 Weekly habit tracker (fully customizable)
- 🥗 Food guide — avoid vs eat more
- 🏃 Exercise guide with countdown timers
- 🧠 Learn section with science-backed tips
- 🔔 Phone notifications
- 💾 Export/Import backup
- 🌙 Dark mode
- 📱 100% offline after first load

## Tech Stack
- Pure HTML, CSS, JavaScript
- PWA (Progressive Web App)
- localStorage for data
- Web Audio API for sounds
- Service Worker for offline + notifications

## How to Deploy
1. Fork this repo
2. Connect to Netlify
3. Done — auto deploys on every commit
